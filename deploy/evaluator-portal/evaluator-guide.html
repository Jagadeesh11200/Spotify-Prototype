Evaluator guide

Does the first song come from habit, or from discovery?

This prototype tests one behavioral question for discovery-frustrated listeners: can Spotify shift the start of a session away from familiar routes by asking for present-moment emotional context before the usual home screen takes over?

Scope

This is a native Android MVP opened through BrowserStack App Live. It uses synthetic listening data, local mock recommendations, and simulated player behavior. What is under test is the interaction model: mood capture, calibrated discovery, player continuity, and adaptive re-entry.

Three phases, in order

Test these in sequence so the behavior change in Phase 3 makes sense.

1
Phase 1

Baseline Spotify shell

Confirm the app still feels like Spotify: Home, Search, Library, bottom navigation, mini player, full player, likes, and familiar content surfaces.

  • Move between Home, Search, and Library using bottom navigation.
  • Start a track and open the full player.
  • Like or unlike a track and confirm the library state responds.
  • Notice that repeat behavior is easy because familiar content is always one tap away.
2
Phase 2

Mood Board into Right Now For You

Review the three-card Mood Board. The center card is the primary choice, while side cards act as alternate emotional worlds.

  • All three mood worlds should feel distinct in color, atmosphere, and fragment language.
  • The center card should show the atmosphere, fragment, memory anchor, and community count clearly.
  • Tap a side card to bring it into the center, then select the center card.
  • Confirm that Home opens with the active mood visible and "Right Now For You" above regular sections.
  • The section header should stay clean. The emotional fragment is not repeated beside it in this build.
  • Play a recommendation and confirm it feeds into the normal player flow.
3
Phase 3

Exit memory and compact entry

This phase tests whether the discovery prompt can back off when a user repeatedly skips it, then return when they actively choose a mood.

  • From the Mood Board, tap "Go to home" without selecting a card five times in a row.
  • On the fifth exit, Home should show a compact persistent mood tile instead of the full intercept.
  • Tap the tile to reopen the full triptych.
  • Select a card and confirm the app reaches "Right Now For You" normally.
  • After selection, the exit count resets and the full Mood Board can return on the next fresh session.

The 90-second script

  1. Open Mobile Session and inspect the three-card Mood Board.
  2. Swap a side card into the center, then select the center card.
  3. Confirm "Right Now For You" appears above normal Home content.
  4. Play a recommendation and open the player.
  5. Return to Mood Board and exit five times to trigger the compact tile.
  6. Tap the tile, select a mood, and confirm the full recommendation flow still works.
